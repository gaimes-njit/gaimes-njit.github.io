To create the uppercase English letter "Q", we will need to use a combination of the block types b11, b31, and b13. The following stream of commands will generate the letter:

```
ab_drop('b31',1)  // horizontal bar of the letter
ab_drop('b13',2)  // vertical bar of the letter
ab_drop('b13',1)  // top left corner of the letter
ab_drop('b13',3)  // top right corner of the letter
ab_drop('b13',4)  // bottom right corner of the letter
ab_drop('b11',2)  // filling the inside of the letter
ab_drop('b11',3)  // filling the inside of the letter
ab_drop('b11',4)  // filling the inside of the letter
```

These commands will construct the uppercase English letter "Q" with the given block types and within the defined environment.
