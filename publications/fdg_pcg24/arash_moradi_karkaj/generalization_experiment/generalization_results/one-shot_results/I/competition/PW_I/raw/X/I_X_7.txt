To generate the uppercase English letter "X" using the given block types, follow the stream of ab_drop() commands below:

```
ab_drop('b13',1)  # Place a vertical straight block at position 1
ab_drop('b31',1)  # Place a horizontal straight block at position 1
ab_drop('b11',2)  # Place a square block at position 2
ab_drop('b31',2)  # Place a horizontal straight block at position 2
ab_drop('b13',3)  # Place a vertical straight block at position 3
```

Please note that these commands should be executed consecutively.
