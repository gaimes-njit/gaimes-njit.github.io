To generate the uppercase letter "U", we can use the following stream of commands:

```
ab_drop('b11',1)  // Place a 1x1 block at position 1
ab_drop('b31',1)  // Place a horizontal straight block at position 1
ab_drop('b13',4)  // Place a vertical straight block at position 4
ab_drop('b31',7)  // Place a horizontal straight block at position 7
ab_drop('b11',7)  // Place a 1x1 block at position 7
```

This series of commands will create a stack of blocks in the shape of an uppercase letter "U" within the allowed space boundaries.
